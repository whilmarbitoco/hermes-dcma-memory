# hermes-dcma-memory

A [Hermes](https://github.com/NousResearch/hermes-agent) memory provider plugin backed by DCMA.

Requires a running DCMA instance. DCMA is private and not publicly available.

## Tools

The plugin exposes Hermes tools for persistent memory — search, storage, retrieval, and consistency checking. Full tool documentation is available in the plugin source.
